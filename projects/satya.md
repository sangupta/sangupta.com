`satya` is a **Social Authentication Library**. The library is nascent and in development. It compiles 
and works for some providers as well.

### Integrated Social Networks

* Google
* Yahoo *broken*
* Facebook
* LinkedIn
* DropBox
* Microsoft Live
* Twitter
* GitHub
